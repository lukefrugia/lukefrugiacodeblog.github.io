What was challenging about Assignment 3:

This was a challenging assignment for me. This was my first time scripting in Javascript, so it was a learning experience for me. The most challenging things I faced when writing Assignment 3 were:

  • Showing the modal.
  • Creating the new post HTML with Javascript.
  • Adding the data attributes to the posts.
  • Adding new posts after learning how to refilter.
  • Refiltering with *all* the posts.

What I learned:

After I found the modal HTML and CSS, I learned how to change the CSS styling with Javascript from this webpage and implemented it in the buttonClickHandler and closePopup functions.

It took me some googling to figure out how to add data attributes when I was creating new posts. Before I starting implementing refiltering, I added these new posts to the posts with getElementById("posts"). This broke because I was deleting everything in the posts div and using an NodeArray that stored the original posts to refilter. It ended up taking a bit of Googling and talking with ChatGPT to figure out that I needed to use Array.from to push a post to the array.
